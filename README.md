# 🏆 Restaurant Analytics & Star Recommendation System

---

## 📌 Project Overview
This project focuses on analyzing restaurant data to identify key factors influencing customer preferences and ratings. The goal is to build a data-driven approach to identify "Star Restaurants" and support recommendation systems for a restaurant aggregator platform.

---

## 🛠️ Tools & Technologies Used
   * **Data Cleaning & Analysis:** Python (Pandas, NumPy)
   * **Visualization:** Matplotlib, Seaborn
   * **Dashboard & Insights:** Tableau Publik

---

## 🔍 Key Steps Performed
 ### 1. Data Cleaning & Preprocessing
  * Removed duplicate records
  * Handled missing values
  * Merged datasets using Country Code
  * Created new features like:
    * Cuisine Count
    * Star Restaurant Flag

---

 ### 2. Exploratory Data Analysis (EDA)
   Performed deep analysis on:
    * 🌍 Geographic distribution of restaurants
    * 🍽️ Top cuisines across cities
    * 🚚 Online delivery trends
    * 📘 Table booking availability
    * ⭐ Rating distribution
    * 💰 Cost distribution
    * 📊 Votes and customer engagement

---

 ### 3. Key Insights
  * Cities like ___ have the highest number of restaurants
  * Online delivery significantly increases customer engagement (votes)
  * Most restaurants fall in the mid-price range
  * Higher cost does not always guarantee higher ratings
  * Popular cuisines dominate customer preference trends

---

 ### 4. Star Restaurant Logic ⭐
   A restaurant is classified as a Star Restaurant based on:
   * Rating > 4.0
   * High number of votes (>100)
   * Availability of online delivery
This helps in identifying top-performing restaurants for recommendations.

---

 ### 5. Tableau Dashboard
   Developed interactive dashboards to visualize:
  * Restaurant distribution by city
  * Top cuisines
  * Cost vs Rating relationship
  * Impact of online delivery
  * Rating influencers
📊 Dashboards Created:
  * Restaurant Overview Dashboard
  * Rating Influencers Dashboard
  * Star Restaurant Dashboard
